# Autonomous Multi-Agent AI System for Business Intelligence

An autonomous, multi-agent pipeline designed to identify digital laggards and generate professional transformation strategies. This system automates the traditionally manual process of business discovery, digital auditing, and client proposal generation.

## 🚀 Core Features
- **Modular Agent Workflow**: Sequential orchestration of 4 specialized agents (Discovery, Analysis, Strategy, Proposal).
- **Digital Maturity Scoring**: Proprietary algorithm for quantifying a business's online footprint.
- **Data Science Evaluation**: Integrated performance benchmarking suite with automated graph generation.
- **Premium Glassmorphism UI**: High-fidelity React dashboard for visualizing system intelligence.
- **Automated ROI Modeling**: ROI-focused strategic synthesis for client acquisition.

## 📊 Performance & Evaluation
The project includes a robust evaluation module to benchmark agentic performance:
```bash
# Run the Moment-by-Moment Analysis
python evaluate_moments.py

# Run the General Performance Benchmarking
python evaluate_performance.py
```
This generates professional visualizations in the `moment_analysis/` and `performance_reports/` directories, analyzing latency, maturity distributions, and systemic opportunity gaps.

## 🛠 Tech Stack
- **Backend**: Python 3.13, FastAPI (Orchestration), SQLAlchemy (ORM), SQLite (Database).
- **Data Science**: Pandas, Matplotlib, Seaborn (Evaluation & Analysis).
- **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion (Glassmorphism Dashboard).

## 🏃‍♂️ Getting Started

### 1. Backend Setup
```bash
cd backend
pip install -r requirements.txt
python main.py
```
Backend runs on `http://localhost:8000`.

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
Frontend runs on `http://localhost:5173`.
